# CS-215 Final Project

## Project Files

### Self Data Analysis Google Colab

File: CS-215 Final Project.ipynb (https://colab.research.google.com/drive/1pwFmUmwu8ji3m8r7ezEHTyquRp-tnoY_?usp=chrome_ntp) 

- This notebook focuses on individual ChatGPT usage analysis. It explores personal conversation patterns, usage frequency, timing trends, and interaction behaviors using data visualization and clustering techniques. The notebook was used to better understand how ChatGPT was utilized over time and across different tasks.

### Main Project Google Colab

File: Combine.ipynb (https://colab.research.google.com/drive/1p56ZvCHkly8PDzFBaJQFLgGiqwWrEhiA?usp=chrome_ntp) 

- This notebook contains the main analysis for the project. It combines datasets from multiple majors, cleans and processes the data, performs clustering and text analysis, and generates visualizations comparing ChatGPT usage patterns between different academic fields such as Computer Science, Physics, Biology, Engineering, and Mathematics.
