# Final Project Ideas

## Overview
This project explores patterns in my personal digital behavior using multiple datasets from tools and platforms I use frequently. Each dataset allows for different types of analysis, ranging from productivity habits to entertainment patterns and strategy-based gameplay.

---

## 1. ChatGPT

### Idea / Topic
I use ChatGPT frequently for a wide range of tasks, making it one of the tools I rely on most in my daily life. This project will explore patterns in how I use ChatGPT, including frequency of use, types of tasks (coding, writing, brainstorming, grammar), and how my usage changes over time.

### Research Questions
- How has my use of ChatGPT changed over time (frequency, prompt length, and complexity)?
- What categories of prompts do I use most frequently?
- How does my usage differ between academic periods and breaks?
- What time of day do I use ChatGPT the most?

### Data Needed
- Chat history (timestamps, prompts, response lengths)
- Prompt categories
- Session frequency and duration

## 4. Rummikub

### Idea / Topic
I have played Rummikub over 3,000 times, making it an interesting dataset for analyzing gameplay strategy and performance.

**Game Overview:**  
Rummikub is a fast-paced, 2–4 player tile-based strategy game where players aim to empty their racks by forming:
- Runs (consecutive numbers, same color)
- Groups (same number, different colors)

Players must first create an initial meld totaling at least 30 points.

### Research Questions
- What strategies lead to higher win rates (e.g., early meld vs. conservative play)?
- Is there a relationship between game duration and winning?
- How does starting tile distribution affect the probability of winning?

### Data Needed
- Game outcomes (win/loss)
- Game duration
- Strategy indicators (e.g., early meld timing)

### Data Source
