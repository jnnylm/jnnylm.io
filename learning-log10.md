# Week 10 Update – Project Setup

## Project Overview

I will be completing this project independently. I am using my personal dataset, and working alone allows me to focus more closely on my own habits and experiences when using ChatGPT.

My project focuses on analyzing my ChatGPT usage patterns. Since this is the AI tool I use most frequently, I want to better understand how my interactions with it evolve over time.

The main goal is to identify patterns in my behavior, including:
- When I use ChatGPT most frequently  
- What types of tasks I use it for  
- How long my prompts usually are  
- How my usage changes depending on context or situation  

---

## Data Source

### ChatGPT Conversation Data

I am using my own ChatGPT conversation history, which I received in JSON format. I will primarily work with the **conversations data**.

### Pros
- Highly relevant to my daily academic and personal workflow  
- Includes both timestamps and text content  
- Allows for deeper behavioral analysis and pattern detection  

### Cons
- Requires significant data cleaning and restructuring  
- Prompt categorization is not built-in and will need to be done manually or semi-manually  

---

## Research Questions

This project will focus on the following questions:

1. How has my use of ChatGPT changed over time?  
2. What types of tasks do I use ChatGPT for most frequently?  
3. What time of day do I use ChatGPT the most?  
4. How does my usage differ during academic periods versus breaks?  
5. How does prompt length relate to response length (e.g., do longer prompts lead to longer responses)?
